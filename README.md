# Tenkiller_Hybrids
Supplemental figures, R code, and input data for reproducibility
For paper submitted for review with NAJFM by White et al. 

Supplemental_Plots_raw_data:  contains supplemental figures as well as some the raw data analyzed in this project.

Supplemental_R_Code_inputs: contains the R code scripts and input files to reproduce the analyses in this project.
